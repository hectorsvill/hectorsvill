# Public Assets

Files in this folder are served from the site root (`/`).

## Résumé

Drop your résumé at `resume.pdf`. The nav bar and the Hero dashboard
both link to it with `download` set, so visitors can save it with a
single click.

If you prefer a different filename, search for `RESUME_URL` in
`client/src/components/HeroDashboard.tsx` and `/resume.pdf` in
`client/src/components/Nav.tsx` + `client/src/components/Footer.tsx`.

## FPV media

Photos and clips live in `fpv/`. See `fpv/README.md` for the drop-in
workflow and `hectorsvill.wiki/Maintenance.md` for the full update
process.
